Welcome! You received a complete Next.js 14 frontend application.

QUICK START (5 minutes):
========================

1. Go to heyama-web folder
   cd heyama-web

2. Install dependencies
   npm install

3. Start the app
   npm run dev

4. Open in browser
   http://localhost:3000

That's it! The app is running.

BEFORE YOU START:
=================

1. Make sure API is running:
   cd ../nestjs-heyama-api
   npm start
   (in another terminal)

2. Check .env.local has:
   NEXT_PUBLIC_API_URL=http://localhost:3000
   NEXT_PUBLIC_SOCKET_URL=http://localhost:3000
