================================================================================
                          HOW TO GET STARTED
================================================================================

STEP 1: READ (5 min)
====================
Start with: 00_START_HERE.txt or GETTING_STARTED.md

STEP 2: INSTALL (3 min)
=======================
cd heyama-web
npm install

STEP 3: CONFIGURE (1 min)
========================
Verify .env.local contains:
  NEXT_PUBLIC_API_URL=http://localhost:3000
  NEXT_PUBLIC_SOCKET_URL=http://localhost:3000

STEP 4: RUN API (if not running)
================================
cd ../nestjs-heyama-api
npm start
(in another terminal)

STEP 5: RUN APP (1 min)
======================
npm run dev

STEP 6: TEST (5 min)
====================
Open: http://localhost:3000
Try: Create, view, delete objects

TOTAL TIME: ~20 minutes from zero to running
